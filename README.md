# 风芽远征

原生 HTML、CSS 与 JavaScript 制作的移动端竖屏 2D Canvas 自动战棋 Demo。

## 在线部署

仓库已配置 GitHub Pages 自动部署：提交并推送到 `main` 分支后，GitHub Actions 会发布 `index.html`。

首次部署前，在 GitHub 仓库中打开 **Settings → Pages**，将 **Build and deployment → Source** 设置为 **GitHub Actions**。

预期访问地址：<https://18668612396.github.io/-BD-/>

也可以在仓库的 **Actions → Deploy GitHub Pages → Run workflow** 中手动触发部署。

## 本地运行

直接用浏览器打开 `index.html` 即可运行。
